Repositori ini berisi simulasi **Anonimisasi Data** sebagai bagian dari tugas mata kuliah Manajemen Data / Keamanan Basis Data. Proyek ini bertujuan untuk mengolah data siswa guna memprediksi risiko putus sekolah dengan tetap menjaga kerahasiaan identitas siswa (*Data Privacy*).

---

## 📂 Struktur Proyek
```text
.
├── data/
│   ├── raw/               # Data mentah sebelum anonimisasi
│   └── processed/         # Data hasil akhir (anonymized)
├── scripts/
│   ├── sql/               # Stored Procedure, View, dan Function
│   └── python/            # Generator data dummy
├── docs/                  # Laporan PDF dan dokumentasi teknis
